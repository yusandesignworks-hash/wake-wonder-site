## タイポグラフィ

### フォントファミリー

## コンポーネントの色使い基準

### CTAボタン（プライマリ）
```css
background: var(--color-accent);
color: var(--color-primary); /* 黒系文字でコントラスト確保 */
```

### ダークセクション（ヒーロー・フッター等）
```css
background: var(--color-bg-dark);
color: var(--color-text-white);
```

### アクセントテキスト（強調）
```css
color: var(--color-accent);
```

### ライトセクション（強みセクション等）
```css
background: var(--color-bg-light);
color: var(--color-text);
```
